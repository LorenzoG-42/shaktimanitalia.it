# Shaktiman Italia Child Theme

Questo è il tema child di Autoev personalizzato per Shaktiman Italia.

## Caratteristiche

- Basato sul tema Autoev di Bravis Themes
- Personalizzazioni specifiche per Shaktiman Italia
- Mantiene la compatibilità con gli aggiornamenti del tema padre

## Installazione

1. Il tema è già installato nella directory wp-content/themes/shaktiman-italia/
2. Attivalo dal pannello di amministrazione WordPress: Aspetto > Temi
3. Personalizza secondo le tue esigenze

## Personalizzazioni

- Aggiungi i tuoi stili CSS nel file style.css
- Aggiungi le tue funzioni PHP nel file functions.php
- Sovrascrivi i template copiandoli dal tema padre e modificandoli

## File importanti

- `style.css` - Stili personalizzati del tema child
- `functions.php` - Funzioni personalizzate del tema child
